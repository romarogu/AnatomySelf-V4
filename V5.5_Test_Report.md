### 5. 数据去重（Idempotency）

| 项目 | 状态 | 验证结果 |
|------|------|----------|
| UNIQUE 索引 | ✅ 通过 | `CREATE UNIQUE INDEX IF NOT EXISTS` 迁移已执行 |
| INSERT OR REPLACE | ✅ 通过 | 使用 `INSERT INTO ... ON CONFLICT DO UPDATE` |
| 批量导入去重 | ✅ 通过 | `batch_add_medical_records` 同样使用 Upsert 逻辑 |

**SQL 实现**:
```sql
INSERT INTO medical_records 
  (profile_id, record_date, indicator_name, value, unit, ...)
VALUES (?, ?, ?, ?, ?, ...)
ON CONFLICT(profile_id, record_date, indicator_name) 
DO UPDATE SET
  value = excluded.value,
  unit = excluded.unit,
  updated_at = CURRENT_TIMESTAMP
```
